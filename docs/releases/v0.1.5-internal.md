# TranFuBar App Internal Test v0.1.5

本版本更新 macOS 内测安装包，并优化顶部菜单栏和启动反馈。

## 主要更新

- 顶部菜单栏默认显示 TranFu 图标、Codex / Claude 图标和剩余额度百分比。
- 修复显示设置已开启但顶部菜单栏样式没有生效的问题。
- App 启动后增加系统通知提醒，提示已在顶部菜单栏运行。
- 更新 macOS 内测安装包。
- 预留在线更新配置，正式签名版后可开启自动更新。

## 使用说明

- 服务器需要拉取最新 `main` 并重启服务。
- 团队成员需要重新下载安装新版 TranFuBar。
- 当前仍是 unsigned 内测包，首次打开仍会有 macOS 安全提醒。

## 下载

安装包见本 Release 的 `TranFuBar-unsigned-test-arm64.zip`。
