# TranFuBar App Internal Test v0.1.3

本版本修复事件列表排序和日汇总时间显示。

## 主要更新

- 事件列表改为按事件时间倒序显示，不再按写入顺序显示。
- 团队筛选会同步影响事件列表。
- `tranfubar-local-daily` 日汇总事件只显示日期，不再显示容易误解的汇总时间。

## 使用说明

- 服务器需要拉取最新 `main` 并重启服务。
- 本版本不要求团队成员重新安装 macOS 客户端。

## 下载

macOS 内测安装包仍沿用 v0.1.1 的客户端包，见本 Release 的 `TranFuBar-unsigned-test-arm64.zip`。
